# 世界的底层理论

## 使用方式

- 用这组元知识帮助用户建立“底层认知框架”，解释不同领域反复出现的现象。
- 面对具体问题时，优先识别相关的约束、能量、信息、反馈、选择、激励、演化、不确定性与结构。
- 不要机械套用所有理论；根据问题挑选最能解释现象的 2–5 个视角，并明确它们如何作用。
- 若用户在做现实决策，帮助其区分事实、假设、概率、成本、激励、反馈回路和长期结果。

## 元知识

## 世界的底层理论

一句话总结：

> 底层理论就是帮助你回答：世界由什么组成？如何变化？信息如何流动？人如何决策？系统为什么稳定或崩溃？什么东西会长期胜出？

如果你想建立自己的“底层认知框架”，可以优先学：
**逻辑学 + 概率论 + 系统论 + 信息论 + 进化论 + 博弈论 + 经济学**。

## 核心

如果压缩成最核心的几个，我会选：

1. **逻辑学**：判断一个说法是否自洽。
2. **概率论**：处理不确定性。
3. **科学方法**：区分观点和可靠知识。
4. **热力学 / 熵**：理解能量、成本、秩序和衰败。
5. **信息论**：理解信息、噪声、沟通和压缩。
6. **系统论**：看整体，而不是只看局部。
7. **控制论**：理解反馈、调节和失控。
8. **进化论**：理解适者生存、选择机制和长期变化。
9. **博弈论**：理解人与人、组织与组织之间的策略互动。
10. **经济学**：理解资源、成本、激励和取舍。

## 详细分层

“底层理论”通常不是某个行业技巧，而是能反复解释不同领域现象的理论。它们关心的是：**约束、能量、信息、反馈、选择、激励、演化、不确定性、结构**。

可以按层次理解：

| 层次 | 底层理论 | 解释什么 |
|---|---|---|
| 思考层 | 逻辑学 | 什么推理是有效的 |
| 思考层 | 数学 | 抽象结构、数量、关系 |
| 认知层 | 科学方法 | 如何获得可靠知识 |
| 认知层 | 概率论 / 贝叶斯 | 如何在不确定中判断 |
| 物理层 | 牛顿力学、相对论、量子论 | 物质与运动 |
| 物理层 | 热力学 / 熵 | 能量、秩序、不可逆变化 |
| 信息层 | 信息论 | 信息、噪声、编码、压缩 |
| 系统层 | 系统论 | 元素、关系、边界、整体性 |
| 系统层 | 控制论 | 反馈、调节、目标、失控 |
| 系统层 | 复杂性理论 | 涌现、非线性、混沌 |
| 生命层 | 进化论 | 适应、竞争、选择、变异 |
| 行为层 | 心理学 / 认知科学 | 人如何感知、决策、犯错 |
| 行为层 | 博弈论 | 多主体互动、合作与冲突 |
| 社会层 | 经济学 | 稀缺、成本、边际、激励 |
| 社会层 | 网络理论 | 连接、传播、中心节点、群体结构 |
| 社会层 | 制度理论 | 规则如何塑造行为 |
| 价值层 | 伦理学 / 政治哲学 | 什么是好的、正当的、公平的 |
